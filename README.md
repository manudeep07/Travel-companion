# Travel Companion – AI-Powered Travel Assistant

Travel Companion is a smart, AI-powered web application built with Flask. It helps travelers by providing real-time weather updates, currency conversions, and concise travel advice using the Gemini API, OpenWeatherMap, and ExchangeRate-API.

A simple and efficient tool for travelers who want quick, intelligent responses without the clutter.

## Features

- AI chatbot that answers travel-related questions using Google's Gemini model
- Real-time weather updates for any city worldwide
- Currency conversion with up-to-date exchange rates
- Lightweight and responsive web app built with Flask
- Cloud-hosted using Render for easy deployment

## Technologies Used

- Python (Flask)
- HTML / CSS / JavaScript
- Google Gemini API
- OpenWeatherMap API
- ExchangeRate-API
- Render (Cloud Hosting)
